# Lazygit チートシート

> Neovim (snacks.nvim) から `<leader>lg` で起動

## グローバル

| キー      | 説明                               |
| --------- | ---------------------------------- |
| `?`       | キーバインド一覧を表示             |
| `q`       | 終了                               |
| `<Esc>`   | キャンセル / 戻る                  |
| `R`       | リフレッシュ                       |
| `z`       | 直前のgitコマンドを取り消し        |
| `Z`       | 取り消しをやり直し                 |
| `:`       | シェルコマンドを実行               |
| `@`       | コマンドログを表示                 |
| `+`       | 画面モード切替（通常/半分/全画面） |
| `{` / `}` | diff表示の前後行数を増減           |
| `<C-w>`   | diffの空白無視を切替               |

## パネル移動

| キー      | 説明                                                     |
| --------- | -------------------------------------------------------- |
| `j` / `k` | リスト内で上下移動 (Status→Files→Branches→Commits→Stash) |
| `h` / `←` | メインパネル → サイドパネルへ戻る                        |
| `l` / `→` | サイドパネル → メインパネルへ移動                        |
| `<Tab>`   | パネル切替（代替キー）                                   |
| `[` / `]` | タブ切替                                                 |
| `<` / `>` | リストの先頭/末尾へ                                      |
| `/`       | 検索                                                     |
| `<Enter>` | 選択 / 展開                                              |

## ファイル操作 (Files パネル)

| キー      | 説明                              |
| --------- | --------------------------------- |
| `<Space>` | ファイルをステージ/アンステージ   |
| `a`       | 全ファイルをステージ/アンステージ |
| `c`       | コミット                          |
| `A`       | 直前のコミットに追加 (amend)      |
| `d`       | 変更を破棄                        |
| `D`       | ワーキングツリーをリセット        |
| `e`       | **エディタで開く (Neovimに戻る)** |
| `o`       | デフォルトアプリで開く            |
| `i`       | .gitignoreに追加                  |
| `s`       | 全変更をstash                     |
| `S`       | stashオプション表示               |
| `g`       | upstream resetオプション          |
| `` ` ``   | ファイルツリー表示を切替          |
| `<C-o>`   | ファイルパスをコピー              |

## ステージング (Staging パネル)

| キー      | 説明                            |
| --------- | ------------------------------- |
| `<Space>` | 行をステージ/アンステージ       |
| `a`       | hunk全体をステージ/アンステージ |
| `d`       | 変更を破棄                      |
| `e`       | **エディタで開く**              |
| `v`       | 範囲選択モード                  |
| `<Esc>`   | ファイルパネルに戻る            |

## ブランチ操作 (Branches パネル)

| キー      | 説明                   |
| --------- | ---------------------- |
| `<Space>` | チェックアウト         |
| `n`       | 新規ブランチ作成       |
| `d`       | ブランチ削除           |
| `r`       | 選択ブランチにリベース |
| `M`       | マージオプション       |
| `R`       | ブランチ名変更         |
| `f`       | fast-forward           |
| `u`       | upstream設定           |
| `c`       | 名前でチェックアウト   |
| `-`       | 直前のブランチに戻る   |
| `g`       | resetオプション        |
| `o`       | PRを作成               |
| `<C-o>`   | ブランチ名をコピー     |

## コミット操作 (Commits パネル)

| キー              | 説明                            |
| ----------------- | ------------------------------- |
| `<Space>`         | detached HEADでチェックアウト   |
| `r`               | コミットメッセージ編集 (reword) |
| `R`               | エディタでreword                |
| `s`               | 下のコミットにsquash            |
| `f`               | 下のコミットにfixup             |
| `d`               | コミットを削除 (drop)           |
| `e`               | コミットを編集 (rebase開始)     |
| `i`               | interactive rebase開始          |
| `p`               | pick (rebase中)                 |
| `t`               | コミットをrevert                |
| `A`               | ステージ内容でamend             |
| `C`               | cherry-pick用にマーク           |
| `V`               | cherry-pickを実行               |
| `T`               | タグを作成                      |
| `n`               | このコミットから新規ブランチ    |
| `g`               | resetオプション                 |
| `<C-j>` / `<C-k>` | コミットを上下に移動 (rebase中) |
| `<C-o>`           | コミットハッシュをコピー        |
| `y`               | コミット情報をコピー            |
| `o`               | ブラウザで開く                  |
| `b`               | bisectオプション                |

## Stash

| キー      | 説明                  |
| --------- | --------------------- |
| `<Space>` | stashを適用           |
| `g`       | stashをpop            |
| `d`       | stashを削除           |
| `n`       | stashから新規ブランチ |
| `r`       | stash名を変更         |

## リモート操作

| キー | 説明  |
| ---- | ----- |
| `P`  | push  |
| `p`  | pull  |
| `f`  | fetch |

## マージ/リベース中

| キー | 説明                       |
| ---- | -------------------------- |
| `m`  | マージ/リベースオプション  |
| `M`  | コンフリクト解決オプション |

---

## Neovim連携 (snacks.nvim)

| Neovimキー   | 説明                     |
| ------------ | ------------------------ |
| `<leader>lg` | Lazygitを開く            |
| `<leader>lf` | 現在ファイルのログを表示 |

Lazygit内で `e` を押すと、Neovimに戻ってファイルが開きます。
