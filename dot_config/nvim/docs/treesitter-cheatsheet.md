# Treesitter チートシート

## Text Objects（選択・操作）

`v`, `d`, `c`, `y` などと組み合わせて使用。

### 関数 (function)

| コマンド | 説明             |
| -------- | ---------------- |
| `vaf`    | 関数全体を選択   |
| `vif`    | 関数の中身を選択 |
| `daf`    | 関数全体を削除   |
| `cif`    | 関数の中身を変更 |
| `yaf`    | 関数全体をヤンク |

### クラス (class)

| コマンド | 説明               |
| -------- | ------------------ |
| `vac`    | クラス全体を選択   |
| `vic`    | クラスの中身を選択 |
| `dac`    | クラス全体を削除   |
| `cic`    | クラスの中身を変更 |

### 引数 (argument)

| コマンド | 説明                         |
| -------- | ---------------------------- |
| `vaa`    | 引数全体を選択（カンマ含む） |
| `via`    | 引数の中身を選択             |
| `daa`    | 引数を削除                   |
| `cia`    | 引数を変更                   |

## 移動

### 関数間の移動

| コマンド | 説明             |
| -------- | ---------------- |
| `]f`     | 次の関数の先頭へ |
| `[f`     | 前の関数の先頭へ |
| `]F`     | 次の関数の末尾へ |
| `[F`     | 前の関数の末尾へ |

### クラス間の移動

| コマンド | 説明               |
| -------- | ------------------ |
| `]c`     | 次のクラスの先頭へ |
| `[c`     | 前のクラスの先頭へ |
| `]C`     | 次のクラスの末尾へ |
| `[C`     | 前のクラスの末尾へ |

### 引数間の移動

| コマンド | 説明       |
| -------- | ---------- |
| `]a`     | 次の引数へ |
| `[a`     | 前の引数へ |

## 実践例

### 関数を別の場所に移動

```
]f      → 次の関数へ移動
daf     → 関数全体を削除（レジスタに保存）
[f      → 移動先へ
P       → 貼り付け
```

### 関数の中身だけ書き換え

```
cif     → 関数の中身を削除してインサートモード
（新しいコードを書く）
<Esc>
```

### 引数の順番を入れ替え

```
daa     → 引数を削除
]a      → 次の引数へ
P       → 貼り付け
```

---

## 覚え方

- `a` = around（周囲を含む）
- `i` = inside（中身のみ）
- `f` = function
- `c` = class
- `a` = argument（text objectの場合）
- `]` = 次へ
- `[` = 前へ
